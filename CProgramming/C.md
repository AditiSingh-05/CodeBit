## Decimal Numbers
A decimal number is a number expressed int the base 10 numeral system consists of o,1,2,3,4,5,6,7,8,9.
Represented by : 

```c
int wholenumber = 25;
int decimalnumber = 3.14;
int largedecimal = 123.456;
```
Example : 

```c
#include <stdio.h>

int main() {
    float num = 12.5;
    printf("The decimal number is: %.2f\n", num);
    return 0;
}
```
